# Interval Timer

シンプルで使いやすいインターバルタイマーのWebアプリケーションです。
筋トレ、HIIT、勉強（ポモドーロテクニック）など、様々な用途に使用できます。

## 特徴

- **カスタムプリセット**: 自分の用途に合わせたタイマー設定を作成・保存できます。
- **詳細な設定**: ワーク/休憩の時間、セット数、繰り返し回数、ロングブレイクなどを自由に設定可能。
- **視覚的なタイマー**: 円形のプログレスバーで残り時間を直感的に把握できます。
- **音声通知**: ステップの切り替えやタイマー完了を音でお知らせします。
- **データ保存**: 設定はブラウザに保存されるため、次回アクセス時もすぐに使えます。
- **PWA対応**: アプリとしてインストールでき、オフラインでも利用可能です。
- **レスポンシブデザイン**: PC、タブレット、スマートフォンなど、様々なデバイスで快適に動作します。
- **バージョン更新**: 画面内のバージョン番号をクリックすることで、最新のリソースに更新できます。

## 使用技術

- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)
- Web Audio API
- Service Worker

## 使い方

1. リポジトリをクローンまたはダウンロードします。
2. `index.html` をブラウザで開きます。
3. 右上の「+」ボタンから新しいプリセットを作成します。
4. 名前、セット内容（ワーク/休憩）、繰り返し回数などを設定して保存します。
5. リストからプリセットを選択し、再生ボタンでタイマーを開始します。

## セットアップ

特別なインストール手順はありません。静的なWebページとして動作します。

```bash
git clone https://github.com/ka-zuu/interval-timer.git
cd interval-timer
# index.htmlをブラウザで開く
```
